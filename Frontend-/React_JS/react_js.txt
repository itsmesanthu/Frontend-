What is React JS?
    • React JS is an open-source JavaScript library developed and maintained by Facebook.
    • It’s widely used for building user interfaces, especially for single-page applications and complex web interfaces.
    • It breaks down a whole website into small, manageable pieces called components.
    • It enables developers to create dynamic, responsive, and reusable UI components, enhancing the efficiency of the development process.
Why React JS?
    • Speed: React uses the Virtual DOM, allowing efficient updates and rendering, resulting in faster and more responsive user interfaces.
    • Reusability: React follows a component-based approach where components can be reused. This leads to cleaner code, easier maintenance, and better collaboration among team members.
    • Job Market: React has high demand. Companies like Netflix, Airbnb, Instagram, etc. use React.
    • Rich Ecosystem and Tooling: React is supported by a vast ecosystem of libraries, tools, and frameworks that enhance its capabilities and help developers build robust applications more efficiently.
About React JS
    • It is a JavaScript library developed by Facebook in 2011. It has been released in 2013.
    • It has been invented by Jordan Walke.
    • Angular JS and Vue JS are the competitors for React.
    • It has Component-Based Architecture, where the component can be reusable which helps in maintaining and organizing the code efficiently.
    • React uses a Virtual DOM to optimize rendering, which improves performance.
    • Unidirectional Data Flow: It means data flows in a single direction, from parent components to child components.
Single Page Application (SPA)
    •⁠  ⁠A Single Page Application (SPA) in React is a web application where the entire website loads only once in the browser. After that, different pages or content are updated dynamically without refreshing the whole page.
