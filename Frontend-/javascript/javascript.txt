======================================== 
JAVASCRIPT FEATURES
========================================

1.  Lightweight JavaScript uses less memory and runs quickly in the
    browser.

2.  Interpreted Language JavaScript does not need a separate compiler.
    The browser reads and executes the code line by line.

3.  Dynamically Typed You do not need to specify the data type.

Example: let x = 10; x = “Hello”;

4.  Cross Platform JavaScript code can run on:

-   Windows
-   Mac
-   Linux
-   Mobile devices

5.  Object-Oriented JavaScript supports OOP concepts like:

-   Objects
-   Classes
-   Inheritance
-   Encapsulation

======================================== 
 VARIABLES 
========================================

A variable is a named memory location used to store data.

Syntax: keyword variableName = value;

Example: let age = 21;

Rules for Variable Names:

1.  Must start with:
    -   Letter
    -   Underscore (_)
    -   Dollar ($)
2.  Cannot start with number

Wrong: let 1age = 21;

3.  Cannot use keywords

Wrong: let let = 10;

4.  Case sensitive

age and Age are different.

----------------------------------------
tpye 1: var 
----------------------------------------
var is the old way to declare variables.
Example: var name = “John”;
Characteristics:

-   Function scope
-   Can redeclare variable
-   Can update value
-   Hoisted to top

Example:

var x = 10; var x = 20;

----------------------------------------
type 2: let 
----------------------------------------

let is modern variable declaration.

Example: let city = “Mumbai”;

Characteristics:

-   Block scope
-   Cannot redeclare
-   Can update value
-   Recommended to use

Example:

let age = 25; age = 30;
----------------------------------------
type 3: const 
----------------------------------------

const is used for constant values.

Example: const PI = 3.14;

Characteristics:

-   Block scope
-   Cannot update value
-   Cannot redeclare
-   Must assign value immediately

Wrong:

const x;

======================================== 
DATA TYPES
========================================

JavaScript has two categories of data types:

1.  Primitive Data Types
2.  Non-Primitive Data Types

  ----------------------
  Primitive Data Types
  ----------------------

1.  String
Stores text.
Example: let name = “Rahul”;

2.  Number
Stores numbers.
Example: let marks = 95;

3.  Boolean
Stores true or false.
Example: let isPassed = true;

4.  Undefined
Variable declared but not assigned.
Example: let x;

5.  Null
Represents empty value.
Example: let data = null;

6.  BigInt
Used for very large numbers.
Example: let bigNumber = 12345678901234567890n;

7.  Symbol
Used to create unique identifiers.
Example: let id = Symbol(“user”);

  --------------------------
  Non-Primitive Data Types
  --------------------------
1.  Object
Stores key-value pairs.
Example:
let person = { name: “John”, age: 25 };

2.  Array
Stores multiple values.
Example:
let numbers = [10, 20, 30];

3.  Function
A block of reusable code.
Example:
function greet() { console.log(“Hello”); }

======================================== 
 TYPEOF OPERATOR 
========================================

typeof is used to check the data type of a variable.

Example:

let age = 25; console.log(typeof age);
Output: number

More Examples:
typeof “Hello” → string typeof true → boolean typeof null → object
typeof [] → object

========================================
 TYPE CONVERSION 
========================================
Type conversion means changing one data type into another.
  ------------------------
  1. Implicit Conversion
  ------------------------
JavaScript automatically converts data types.

Example:
let result = “5” + 2;
Output: 52

  ------------------------
  2. Explicit Conversion
  ------------------------
Programmer manually converts data type.

String to Number:
let num = Number(“100”);

Number to String:
let str = String(100);

Boolean to Number:
let num = Number(true);
Output: 1

• JavaScript is an object-based language.
• An object is a combination of properties and behavior (methods).
• An object represents a real-world entity.
• JavaScript is template-based, not purely class-based.

name and age are keys

Object.assign()
-----------------
- Object.assign() is a built-in JavaScript method.
- It is used to copy properties from one or more source objects to a target object.
- It merges objects together into a single object.
- It returns the modified target object.

MAIN PURPOSE

1. Copy properties from one object to another
2. Merge multiple objects
3. Create a shallow copy of an object

IMPORTANT POINTS

- It copies only enumerable properties.
- It performs a shallow copy (not deep copy).
- If properties have the same name, the last value overwrites the previous one.
- The target object gets updated.

SYNTAX

Object.assign(target, source1, source2, ...)

TERMS

Target:
The object that receives the properties.

Source:
The object(s) from which properties are copied.

Object.entries()
---------------------
- Object.entries() is a built-in JavaScript method.
- It returns all the properties of an object.
- The result is returned as an array.
- Each element in the array is a pair of key and value.
- The pair is stored as a sub-array.

PURPOSE

- To get both keys and values together.
- To convert an object into an array format.
- Useful for looping through object data.

RETURN TYPE

- Returns an array of key-value pairs.

SYNTAX

Object.entries(object_name)

IMPORTANT POINTS

- It includes only enumerable properties.
- The order of entries follows the order of properties in the object.
- It does not change the original object.

Object.freeze()
--------------------
- Object.freeze() is a built-in JavaScript method.
- It is used to completely lock an object.
- It prevents adding new properties.
- It prevents deleting existing properties.
- It prevents modifying existing property values.
- The object becomes read-only.

PURPOSE

- To protect an object from any changes.
- To make object data constant.
- To ensure data safety.

SYNTAX

Object.freeze(object_name)

IMPORTANT POINTS

- No new properties can be added.
- Existing properties cannot be deleted.
- Existing property values cannot be changed.
- It returns the same frozen object.
- It performs a shallow freeze (nested objects can still change).

=========================
JAVASCRIPT ARRAY
=========================
- JavaScript Array is a special variable used to store multiple values in a single variable.
- Arrays store data in an ordered list.
- Each value in an array is called an element.
- Each element has an index number.
- Index numbering starts from 0.

CHARACTERISTICS OF ARRAY

1. Arrays can store multiple values.
2. Arrays can store different data types.
3. Arrays are dynamic (size can change).
4. Arrays are zero-indexed.
5. Arrays are objects in JavaScript.

ARRAY CREATION METHODS

1. Using Array Literal
2. Using new Array() constructor

ARRAY SYNTAX
-------------------------

1. Array Declaration

   var array_name;

2. Array Initialization (Array Literal)

   var array_name = [value1, value2, value3];

3. Array Using Constructor

   var array_name = new Array(value1, value2, value3);

4. Accessing Array Element

   array_name[index];

5. Updating Array Element

   array_name[index] = new_value;

6. Finding Array Length

   array_name.length;

ARRAY INDEX

- Index is the position of an element in an array.
- First element index is 0.
- Second element index is 1.
- Last element index is length - 1.

ARRAY PROPERTIES

length
- Returns the number of elements in the array.

ARRAY COMMON OPERATIONS

1. Add element
2. Remove element
3. Access element
4. Update element
5. Traverse element

COMMON ARRAY METHODS

push()
- Adds element at the end of array.

pop()
- Removes element from the end.

shift()
- Removes element from the beginning.

unshift()
- Adds element at the beginning.

slice()
- Returns a portion of the array.

splice()
- Adds or removes elements from array.

indexOf()

includes()
- Checks if element exists.

join()

reverse()
- Reverses the array.

sort()
- Sorts the array elements.

forEach()

map()
- Creates new array after applying function.

filter()
- Returns elements that match condition.

reduce()
