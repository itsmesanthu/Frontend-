==================================================================================================================================================
                JAVASCRIPT

======================================== 
JAVASCRIPT FEATURES
========================================

1.  Lightweight JavaScript uses less memory and runs quickly in the
    browser.

2.  Interpreted Language JavaScript does not need a separate compiler.
    The browser reads and executes the code line by line.

3.  Dynamically Typed You do not need to specify the data type.

Example: let x = 10; x = “Hello”;

4.  Cross Platform JavaScript code can run on:

-   Windows
-   Mac
-   Linux
-   Mobile devices

5.  Object-Oriented JavaScript supports OOP concepts like:

-   Objects
-   Classes
-   Inheritance
-   Encapsulation

======================================== 
 VARIABLES 
========================================

A variable is a named memory location used to store data.

Syntax: keyword variableName = value;

Example: let age = 21;

Rules for Variable Names:

1.  Must start with:
    -   Letter
    -   Underscore (_)
    -   Dollar ($)
2.  Cannot start with number

Wrong: let 1age = 21;

3.  Cannot use keywords

Wrong: let let = 10;

4.  Case sensitive

age and Age are different.

----------------------------------------
tpye 1: var 
----------------------------------------
var is the old way to declare variables.
Example: var name = “John”;
Characteristics:

-   Function scope
-   Can redeclare variable
-   Can update value
-   Hoisted to top

Example:

var x = 10; var x = 20;

----------------------------------------
type 2: let 
----------------------------------------

let is modern variable declaration.

Example: let city = “Mumbai”;

Characteristics:

-   Block scope
-   Cannot redeclare
-   Can update value
-   Recommended to use

Example:

let age = 25; age = 30;
----------------------------------------
type 3: const 
----------------------------------------

const is used for constant values.

Example: const PI = 3.14;

Characteristics:

-   Block scope
-   Cannot update value
-   Cannot redeclare
-   Must assign value immediately

Wrong:

const x;

======================================== 
DATA TYPES
========================================

JavaScript has two categories of data types:

1.  Primitive Data Types
2.  Non-Primitive Data Types

  ----------------------
  Primitive Data Types
  ----------------------

1.  String
Stores text.
Example: let name = “Rahul”;

2.  Number
Stores numbers.
Example: let marks = 95;

3.  Boolean
Stores true or false.
Example: let isPassed = true;

4.  Undefined
Variable declared but not assigned.
Example: let x;

5.  Null
Represents empty value.
Example: let data = null;

6.  BigInt
Used for very large numbers.
Example: let bigNumber = 12345678901234567890n;

7.  Symbol
Used to create unique identifiers.
Example: let id = Symbol(“user”);

  --------------------------
  Non-Primitive Data Types
  --------------------------
1.  Object
Stores key-value pairs.
Example:
let person = { name: “John”, age: 25 };

2.  Array
Stores multiple values.
Example:
let numbers = [10, 20, 30];

3.  Function
A block of reusable code.
Example:
function greet() { console.log(“Hello”); }

======================================== 
 TYPEOF OPERATOR 
========================================

typeof is used to check the data type of a variable.

Example:

let age = 25; console.log(typeof age);
Output: number

More Examples:
typeof “Hello” → string typeof true → boolean typeof null → object
typeof [] → object

========================================
 TYPE CONVERSION 
========================================
Type conversion means changing one data type into another.
  ------------------------
  1. Implicit Conversion
  ------------------------
JavaScript automatically converts data types.

Example:
let result = “5” + 2;
Output: 52

  ------------------------
  2. Explicit Conversion
  ------------------------
Programmer manually converts data type.

String to Number:
let num = Number(“100”);

Number to String:
let str = String(100);

Boolean to Number:
let num = Number(true);
Output: 1

Named Function:
----------------
Definition:
A named function is a function defined using the function keyword
with a specific name.

Advantages of Named Function:
    1. Easy to reuse the function.
    2. Improves code readability.
    3. Can be called multiple times.

How to create a Named function in JS?
    * Use the function keyword.
    * Provide a function name.
    * Call the function to execute.

Syntax:
function functionName() {
    // code to be executed
}


Parameterized Function:
-----------------------
Definition:
A parameterized function is a function that accepts input values called parameters.
Advantages of function:
    1.Dynamic input handling
    2.Code flexibility
How to create a function in JS?
    * Define parameters inside parentheses
    * Pass values while calling the function
Syntax:
    function functionName(parameter1, parameter2) {
        // code
    }

Return funciton:
-----------------
Definition:
A return function is a function that sends a value back using the return keyword.
Advantages of function:
    1. Returns result to caller
    2. Can store output in variables

How to create a function in JS?
    * Use return keyword
    * Store the returned value
    * ofter using return it belowe code is not run 

Syntax:
    function functionName() {

    }

Anonymous Function:
---------------------
Definition:
An anonymous function is a function without a name.

Advantages of function:
    1. Useful for temporary operations
    2. Used in callbacks and events
How to create a function in JS?
    * Assign function to a variable
Syntax:
    var variableName = function() {
          // code
    }

Callback Function:
---------------------
Definition:
A callback function is a function passed as an argument to another function.

Advantages of function:
    1. Enables asynchronous programming
    2. Improves flexibility
How to create a function in JS?
    * Pass a function as parameter
Syntax:
    function mainFunction(callback) {
        callback()
    }

Arrow Function:
----------------
Definition:
An arrow function is a shorter syntax for writing functions using =>.
Advantages of function:
    1. Shorter syntax
    2.Cleaner code
How to create a function in JS?
    * Use arrow symbol =>
Syntax:
    const functionName = (parameters) => {
        // code
    }

Immediately Invoked Function (IIFE):
--------------------------------------
Definition:
An IIFE is a function that runs immediately after it is defined.
Advantages of function:
    1. Runs automatically
    2. Avoids global variable pollution
How to create a function in JS?
    * Wrap function in parentheses
    * Add () to invoke
Syntax:
    (function() {
        // code
    })();

Currying Function:
---------------------
Definition:
A currying function is a function that returns another function.
Advantages of function:
    1. Breaks complex functions
    2. Improves reusability
How to create a function in JS?
    * Return a function inside another function
Syntax:
    function functionName(a) {
        return function(b) {
            // code
        }
    }
